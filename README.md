# Dissscord

Full-stack clone of Discord written in Python (üêç hence the name diSSScord................................üò≠)The backend is built using Flask and SQLAlchemy and the frontend is rendered using React. Click [here](https://github.com/aznguymp4/Dissscord/wiki) to view our database schema. 


To view the backend routes, click [here](https://dissscord.onrender.com/api/docs)

## Technologies Used
- React.js
- Flask
- SQLAlchemy
- PostgreSQL
- Render
- Socket.io
- Javascript
- Python
- Css
- HTML

## Features
 - Users can create, update, and delete servers
 - Users can create, update, and delete channels for servers
 - Users can create, update, and delete messages in servers	
 - Users can add and delete reactions to messages
 - Users can join and leave public servers
